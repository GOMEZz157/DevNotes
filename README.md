# 📝 DevNotes

DevNotes é uma aplicação web simples para **criar, organizar e gerenciar notas diretamente no navegador**.

O projeto foi desenvolvido com foco em produtividade, permitindo criar, editar e salvar anotações rapidamente. Todas as notas ficam armazenadas no **LocalStorage**, garantindo que permaneçam salvas mesmo após fechar o navegador.

Além disso, o sistema permite **exportar todas as notas para um arquivo CSV**, facilitando backup ou uso em outras ferramentas.

---

## 🚀 Funcionalidades

- ➕ Criar novas notas  
- ✏️ Editar notas existentes  
- 📌 Fixar notas importantes no topo  
- 🗑️ Excluir notas  
- 💾 Salvamento automático com **LocalStorage**  
- 📤 Exportar notas para **CSV**  
- 📱 Interface responsiva  

---
